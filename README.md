# capistrano-docker-compose

Docker Compose integration for Capistrano.

## Features

- Standard Capistrano code deploys
- Environment-specific Docker container deploys

## Requirements

- Capistrano 3.4+
- Docker Engine 1.11+
- Docker Compose 1.7+
- Phusion Passenger 5+
